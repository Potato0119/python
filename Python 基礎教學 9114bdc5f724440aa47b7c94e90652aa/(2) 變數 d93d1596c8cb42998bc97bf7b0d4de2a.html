(2) 變數

變數宣告

變數是程式語言裡重要的一環

沒有他什麼都沒有

變數就是一個可以變動的數值 (不是數字)

所以

name = "小帥"

這就是一個變數

用中文來說 就是 一個叫做name的變數 他的值是小帥

左邊是變數名 中間一個等於(表達賦予之意) 右邊是值

可以說是小帥賦予name一個值

這時 要結合第一單元所學的資料型態( 字元 字串 整數 浮點數 布林值 )

我每個都舉個例子

字元 letter = "a"

字串 name = "Thomas Cruise"

整數 num = 12

浮點數 distance = 5.634

布林值 isRun = Ture

都是同一個規律 左邊名稱 右邊值

補充 變數取名

  • 字首不可為數字 但非字首可以
  • 大小寫有區別　除了英文單字還可活用 底線 _
  • 另外不能取名到保留字 (不能使用這些來當變數名, 共35個)

變數的加減乘除

字串

字串也是可以做加法跟乘法的

假如說有

content1 = "Hello"
content2 = "World"
content3 = content1+content2

那content1加上content2 就是Hello加上World

也就是HelloWorld 因此content3 = HelloWorld

x="5"
y="10"
z=x+y

z是510　不是15

因為字串5 + 字串10

5 + 10 = 510

所以不要搞混字串與整數了

content = "Hello"
x = 3
result = content*x

你們應該在思考這是什麼鬼

為什麼字串可以跟整數乘 難道不會報錯嗎

不會 可以想想前面"Hello" + "Hello" = "HelloHello"

2+2+2=2*3對吧

所以把字串乘3就是這樣的概念

所以result = "HelloHelloHello"

(據我所知 應該只有Python跟少部分的其他程式語言可以這樣使用 這個部分要小心一點)

整數

整數就相當簡單了

  • +就加
  • -就是減
  • *就是乘
  • /就是除

就這樣　沒什麼難的 更複雜的運算還在後面的單元　運算子

num1 = 6
num2 = 3

result1 = num1 + num2
result2 = num1 - num2
result3 = num1 * num2
result4 = num1 / num2

相信國小有畢業都會

result1 = 9

result2 = 3

result3 = 18

result4 = 2
