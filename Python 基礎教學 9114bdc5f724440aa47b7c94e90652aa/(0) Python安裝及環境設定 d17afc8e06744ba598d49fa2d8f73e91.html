(0) Python安裝及環境設定

要學Python就要先裝Python

這是廢話

所以先到 https://www.python.org/

依照上面的方法下載 先按download 然後選擇Python 3.10.2 (或是最新版的 都可以)

下載好之後 找到他 然後把它點兩下打開

先按下箭頭的 Add python.exe to PATH

把它打勾之後 按上方的Install Now

然後給它跑 好了之後會長這樣

就代表成功了 按下箭頭所指的Close

安裝整合開發環境

但你以為只有這麼簡單喔 安裝好這個你要有地方寫啊

你當然也可以打開記事本像個87慢慢寫

或是用他那個難用到我差點流眼淚的IDLE (考證照要用那個 哭死)

或是你有更好的選擇 選擇一個十分優秀的整合開發環境(IDE)

所以在此 我推薦 Visual Studio Code (簡稱VS code)作為你的不二之選

https://code.visualstudio.com/

把這個Download按鈕按下去就對了

然後他就會下載

如果瀏覽器提示你 這個軟體可能會損害你的裝置 請選擇保留檔案

它並不會帶給你的電腦任何的傷害

接下來的安裝我就不想再丟圖片了 就是 我同意→下一步→下一步→安裝

然後說安裝成功就關掉

然後找到軟體本人 打開它

接著你會發現一片英文 根本看不懂

所以請照我的步驟 依序點開

在2的地方輸入Chinese

就會有繁體中文的延伸套件可以安裝

(我已經安裝了 所以他會顯示解除安裝)

安裝好之後 回到搜尋欄 輸入Python

然後找到該套件 安裝

等它安裝成功後 把Visual Studio Code關掉

再重開 這樣就配置好你的整合開發環境了

建立放檔案的地方

重新打開後還是一臉問號

到底什麼時候才能寫到程式

其實就差一步而已了

看到左邊的 開啟資料夾嗎

先別點

你可以在桌面或是任何你記得的地方 新增一個資料夾 名字可以亂取

然後這個資料夾以後就是你放這些程式碼範例的地方了

接著在按 開啟資料夾

依照路徑找到你新增的資料夾 開啟

這樣就終於配置好了

接著按下箭頭所指方向 這是新增檔案的圖標

按一下 取檔名 副檔名要 “.py”

檔案名稱.py

舉例: main.py first.py 之類的

檔名快樂取 但請注意一句話 現在可能聽不懂 但以後我會再提醒一次

不要取跟任何一個函式或函式庫一樣的名字

為了避免這種悲劇 我建議一律都叫main.py

main 主要的 adj
